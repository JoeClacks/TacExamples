Some example projects to help learn modding of Kerbal Space Program.


For use with the Kerbal Space Program, http://kerbalspaceprogram.com/

Licensed under the Apache License Version 2.0.  See the LICENSE.txt and NOTICE.txt files.

Source code can be found at https://github.com/taraniselsu/TacExamples


===== Features =====
* 


- Show how to make a part-less plugin (KSPAddon)
- Show how to make a PartModule plugin
- Show how to use KSPField's
- Show how to use KSPAction's
- Show how to use KSPEvent's
- Show how to make a simple GUI
- define a new resource


===== Installation instructions =====
* Unzip to the KSP directory.
