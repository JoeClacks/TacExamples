Some example projects to help learn modding of Kerbal Space Program.

Inspired by evilC and his work with my TacLib:
 https://github.com/evilC/TacLib

For use with the Kerbal Space Program, http://kerbalspaceprogram.com/

Licensed under the Apache License Version 2.0.  See the LICENSE.txt and
NOTICE.txt files.

Source code can be found at https://github.com/taraniselsu/TacExamples


===== Features =====
* Includes working example projects for:
	- A simple part-less plugin.
	- A simple PartModule plugin.
	- A part with right-click events.
	- A part with action group events.

* Coming soon(TM):
	- Show how to use KSPField's
	- Show how to make a simple GUI
	- define a new resource


===== How to use =====
* Read over the comments and code to learn how to add a feature to
	another mod.
* Use one of the examples as a starting point for a new mod.
* ...?
* Profit!
